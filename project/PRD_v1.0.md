**正文：**
今天起，当有人搜索你的名字或扫描你的二维码，他们看到的不再是空白或社交平台主页，而是一个清晰、专业、有温度的个人品牌页面。页面在5秒内传达核心价值，在30秒内完成信任建立，并提供一个明确的行动入口（联系/合作/下载简历）。无需注册，无需登录，打开即用。

## 2. 用户画像

### 主要受众：「第一次接触者」
- **场景**：收到名片/链接，想快速了解对方是谁
- **设备**：60%移动端，40%桌面端
- **停留时长预期**：15–90秒
- **核心问题**：「这个人/团队能帮我解决什么问题？」「我该怎么联系他？」
- **痛点**：信息散落在多个平台，找不到一个权威入口

### 次要受众：「搜索引擎来访者」
- **场景**：搜索名字或关键词后点进来
- **需求**：快速判断是否值得深入了解

## 4. 信息架构（IA）

```
单页滚动结构（Single Page Scroll）

├── [Section 1] Hero / 首屏
│   ├── 头像/Logo
│   ├── 姓名 + 一句话定位（Tagline）
│   ├── 副标题（3个关键词/能力标签）
│   └── CTA按钮：「联系我」→ 锚点跳转到联系区
│
├── [Section 2] 关于我 / About
│   ├── 2-3段简介文字
│   └── 技能/标签云（可选）
│
├── [Section 3] 作品/服务 / Work & Services
│   ├── 卡片式布局（3-6张）
│   ├── 每张：封面图 + 标题 + 一句描述 + 链接
│   └── 「查看更多」（可选，链接到外部）
│
├── [Section 4] 数据/成就 / Numbers（可选）
│   └── 3个核心数字（如：服务客户数/项目数/年限）
│
├── [Section 5] 联系 / Contact
│   ├── 联系表单（姓名/邮箱/留言 → Formspree）
│   ├── 社交媒体图标链接
│   └── 邮箱直链
│
└── [Footer]
    ├── 版权信息
    └── 备案号（如需）
```

## 7. 北极星指标 + 指标体系

### 北极星指标
**联系转化率** = 提交联系表单人数 / 页面UV  
> 目标：≥ 3%（行业基准1-2%，我们要做到1.5-2倍）

### 一级指标（健康度）
| 指标 | 定义 | 目标值 |
|------|------|--------|
| 页面UV | 独立访客数 | 基准期建立后设目标 |
| 平均停留时长 | 用户在页面停留秒数 | ≥ 45秒 |
| 跳出率 | 只看首屏就离开的比例 | ≤ 60% |
| 联系表单提交数 | 绝对转化量 | ≥ 5次/月（冷启动） |

### 二级指标（诊断用）
| 指标 | 定义 | 用途 |
|------|------|------|
| 首屏CTA点击率 | 点击「联系我」按钮的比例 | 诊断Hero区吸引力 |
| 作品区点击率 | 点击任意作品卡片的比例 | 诊断内容吸引力 |
| 移动端占比 | 移动端UV/总UV | 指导响应式优先级 |
| 页面加载时间 | LCP ≤ 2.5s | 技术健康度 |
| 社交链接点击率 | 各平台图标点击比例 | 了解用户偏好渠道 |

## 8. 技术约束与非功能需求

| 项目 | 要求 |
|------|------|
| 页面体积 | 首屏资源 ≤ 200KB（不含图片） |

| 无障碍 | WCAG 2.1 AA，alt文字，键盘可导航 |

| HTTPS | 必须，GitHub Pages/Vercel默认支持 |
| 无Cookie弹窗 | 不接需要GDPR弹窗的服务 |
